# Security Policy

## Supported Versions

Currently, only the latest version of Lite is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please report it privately.

**DO NOT create a public issue for security-related reports.**

Please email security reports directly to: **csv610@gmail.com**.

Include a detailed description of the vulnerability, steps to reproduce it, and any potential fix if available. We will respond to your report as soon as possible and work towards a resolution.
