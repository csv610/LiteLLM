# SANE Interview

This package contains a trauma-informed interview workflow for Sexual Assault Nurse Examiner contexts.

## Files

- `sane_interview.py`: interview flow.
- `sane_interview_cli.py`: CLI entrypoint used by `medkit-sane`.
- `sane_interview_chatbot.py`: conversational interface.
- `sane_interview_models.py`: models.
- `llm_sane_interview.py`: related helper.

## Why It Matters

This module focuses on structured questioning in a sensitive context where tone and sequencing matter.

## Limitations

- The tool does not replace trained forensic or clinical personnel.
- Deployment requires careful review for trauma-informed and legal appropriateness.
