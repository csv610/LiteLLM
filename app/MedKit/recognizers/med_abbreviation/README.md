# Medical Abbreviation Recognizer

This module identifies and expands medical abbreviations in text.

## Files

- `medical_abbreviation_recognizer.py`: recognition logic.
- `medical_abbreviation_cli.py`: CLI interface.
- `medical_abbreviation_models.py`: schemas.
- `medical_abbreviation_prompts.py`: prompts.

## Limitations

- Many abbreviations are ambiguous and require local context.
