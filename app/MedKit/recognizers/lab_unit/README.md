# Lab Unit Recognizer

This module identifies laboratory units and related value expressions in text.

## Files

- `lab_unit_recognizer.py`: recognition logic.
- `lab_unit_cli.py`: CLI interface.
- `lab_unit_models.py`: schemas.
- `lab_unit_prompts.py`: prompts.

## Limitations

- Units and values can be separated or ambiguously formatted in clinical text.
