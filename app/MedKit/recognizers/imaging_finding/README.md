# Imaging Finding Recognizer

This module identifies imaging-related findings in text.

## Files

- `imaging_finding_recognizer.py`: recognition logic.
- `imaging_finding_cli.py`: CLI interface.
- `imaging_finding_models.py`: schemas.
- `imaging_finding_prompts.py`: prompts.

## Limitations

- Findings can be modality-specific and phrased in many ways.
