# Genetic Variant Recognizer

This module identifies genes, variants, and related genomic mentions in text.

## Files

- `genetic_variant_recognizer.py`: recognition logic.
- `genetic_variant_cli.py`: CLI interface.
- `genetic_variant_models.py`: schemas.
- `genetic_variant_prompts.py`: prompts.

## Limitations

- Variant nomenclature can be complex and context-sensitive.
