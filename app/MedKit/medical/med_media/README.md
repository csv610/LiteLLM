# Medical Media

This module searches or processes medical media resources such as images and videos.

## Files

- `med_media.py`: main workflow.
- `med_media_cli.py`: CLI interface.
- `med_media_models.py`: schemas.
- `med_media_prompts.py`: prompts.
- helper scripts for image and video retrieval.

## Limitations

- Media availability and search results depend on external services.
- Retrieved media should be checked for provenance and usage rights.
