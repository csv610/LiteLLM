# Medical Anatomy

This module generates structured descriptions of anatomical structures and includes an evaluation helper for generated reports.

## Files

- `medical_anatomy.py`: anatomy generator.
- `medical_anatomy_cli.py`: CLI interface.
- `medical_anatomy_models.py`: schemas.
- `medical_anatomy_prompts.py`: prompts.
- `evaluate_anatomy_report.py`: report evaluation helper.
- `evaluate_anatomy_cli.py`: evaluator CLI.

## Limitations

- Anatomical detail may be incomplete or oversimplified.
- Generated reports should be reviewed for educational accuracy.
