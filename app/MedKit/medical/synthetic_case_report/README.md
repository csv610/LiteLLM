# Synthetic Case Report

This module generates synthetic medical case reports for educational or testing workflows.

## Files

- `synthetic_case_report.py`: generation logic.
- `synthetic_case_report_cli.py`: CLI interface.
- `synthetic_case_report_models.py`: schemas.
- `synthetic_case_report_prompts.py`: prompts.

## Limitations

- Generated cases are fictional and may not reflect realistic prevalence or complexity.
- They should not be mistaken for real patient records.
