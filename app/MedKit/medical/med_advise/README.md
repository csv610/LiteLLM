# Medical Advice Support

This module generates general primary-care guidance and self-care-oriented educational output.

## Files

- `primary_health_care.py`: generation logic.
- `primary_health_care_cli.py`: CLI interface.
- `primary_health_care_models.py`: schemas.
- `primary_health_care_prompts.py`: prompts.

## Limitations

- It provides general guidance, not patient-specific medical advice.
- Urgent or high-risk symptoms require real clinical evaluation.
