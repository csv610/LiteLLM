# Medical Procedure Information

This package contains both procedure-description generation and procedure-output evaluation tools.

## Files

- `medical_procedure_info.py`: procedure information generator.
- `medical_procedure_info_cli.py`: generator CLI.
- `medical_procedure_info_models.py`: schemas.
- `medical_procedure_info_prompts.py`: prompts.
- `eval_medical_procedure_output.py`: evaluator logic.
- `eval_medical_procedure_output_cli.py`: evaluator CLI.

## Limitations

- Procedural descriptions are educational summaries.
- Generated or evaluated procedure reports should be reviewed by qualified users.
