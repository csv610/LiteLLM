# Surgical Tray

This module generates structured tray setups for named surgical contexts.

## Files

- `surgical_tray_info.py`: generation logic.
- `surgical_tray_info_cli.py`: CLI interface.
- `surgical_tray_info_models.py`: schemas.
- `surgical_tray_info_prompts.py`: prompts.

## Limitations

- Tray composition varies by institution and surgeon preference.
- Output should be treated as educational rather than operational setup guidance.
