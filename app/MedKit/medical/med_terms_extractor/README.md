# Medical Terms Extractor

This module extracts medical terms from text and returns them in a structured format.

## Files

- `medical_term_extractor.py`: extraction logic.
- `medical_term_extractor_cli.py`: CLI interface.
- `medical_term_extractor_models.py`: schemas.
- `medical_term_extractor_prompts.py`: prompts.

## Limitations

- Extraction quality depends on text quality and model behavior.
- Term boundaries and categorization may require manual review.
