# Surgical Pose Information

This module generates structured descriptions of surgical positioning and related risks.

## Files

- `surgical_pose_info.py`: generation logic.
- `surgical_pose_info_cli.py`: CLI interface.
- `surgical_pose_info_models.py`: schemas.
- `surgical_pose_info_prompts.py`: prompts.

## Limitations

- Positioning guidance should be checked against current perioperative standards.
