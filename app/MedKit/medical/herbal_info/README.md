# Herbal Information

This module generates structured descriptions of herbal products and related claims.

## Files

- `herbal_info.py`: generation logic.
- `herbal_info_cli.py`: CLI interface.
- `herbal_info_models.py`: schemas.
- `herbal_info_prompts.py`: prompts.

## Limitations

- Evidence quality for herbal products varies substantially.
- Safety, interactions, and regulatory status should be independently verified.
