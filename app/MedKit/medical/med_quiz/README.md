# Medical Quiz

This module generates medical multiple-choice questions and related quiz content.

## Files

- `medical_quiz.py`: generation logic.
- `medical_quiz_cli.py`: CLI interface.
- `medical_quiz_models.py`: schemas.
- `medical_quiz_prompts.py`: prompts.

## Limitations

- Question quality and distractor quality depend on the model.
- Generated quizzes should be reviewed before educational use.
