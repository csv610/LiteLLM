# Medical Flashcard

This module extracts and explains terms from a medical label or image.

## Files

- `medical_flashcard.py`: extraction and explanation logic.
- `medical_flashcard_cli.py`: CLI interface.
- `medical_flashcard_models.py`: schemas.
- `medical_flashcard_prompts.py`: prompts.

## Limitations

- Image quality and OCR-like interpretation affect output quality.
- Extracted explanations should be reviewed for correctness.
