# MedKit Medical

This package groups the general medical-information modules used by `medkit-medical`.

## What It Includes

- anatomy and disease reference modules
- clinical support modules such as advice, decision guides, FAQ, facts, and referral
- educational modules such as quiz, flashcards, and synthetic cases
- procedure, surgery, tool, tray, and positioning modules

## Entry Point

- `medical_cli.py`: unified CLI used by `medkit-medical`

## Why It Matters

These modules cover a wide range of medical-information tasks under a single command namespace.

## Limitations

- The package is broad but not clinically validated.
- Outputs should be reviewed before educational or operational use.
