# Medical Facts Checker

This module evaluates a medical statement and generates a structured assessment of its plausibility or correctness.

## Files

- `medical_facts_checker.py`: checking logic.
- `medical_facts_checker_cli.py`: CLI interface.
- `medical_facts_checker_models.py`: schemas.
- `medical_facts_checker_prompts.py`: prompts.

## Limitations

- Fact checking by an LLM is not equivalent to source-based verification.
- Claims should still be checked against primary references.
