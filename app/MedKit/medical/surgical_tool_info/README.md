# Surgical Tool Information

This module generates structured descriptions of surgical tools and instruments.

## Files

- `surgical_tool_info.py`: generation logic.
- `surgical_tool_info_cli.py`: CLI interface.
- `surgical_tool_info_models.py`: schemas.
- `surgical_tool_info_prompts.py`: prompts.

## Limitations

- Instrument naming and usage can vary across settings and specialties.
