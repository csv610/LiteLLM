# Organ Diseases

This module generates structured information about diseases associated with a named organ.

## Files

- `organ_disease_info.py`: generation logic.
- `organ_disease_info_cli.py`: CLI interface.
- `organ_disease_info_models.py`: schemas.
- `organ_disease_info_prompts.py`: prompts.

## Limitations

- Organ-level grouping is a simplification and may miss systemic disease relationships.
