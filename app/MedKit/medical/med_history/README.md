# Medical History

This module generates structured history-taking questions for a selected exam or visit context.

## Files

- `patient_medical_history.py`: generation logic.
- `patient_medical_history_cli.py`: CLI interface.
- `patient_medical_history_models.py`: schemas.
- `patient_medical_history_prompts.py`: prompts.

## Limitations

- Intake questions should be adapted to local practice and patient context.
- The module assists question generation, not diagnostic reasoning.
