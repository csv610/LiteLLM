# Medical Implant

This module generates structured descriptions of medical implants and related considerations.

## Files

- `medical_implant.py`: generation logic.
- `medical_implant_cli.py`: CLI interface.
- `medical_implant_models.py`: schemas.
- `medical_implant_prompts.py`: prompts.

## Limitations

- Implant specifications and approved indications can change over time.
- Output should be verified against current manufacturer and regulatory sources.
