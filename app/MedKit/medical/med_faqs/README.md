# Medical FAQ

This module generates patient-facing frequently asked questions for a medical topic.

## Files

- `medical_faq.py`: generation logic.
- `medical_faq_cli.py`: CLI interface.
- `medical_faq_models.py`: schemas.
- `medical_faq_prompts.py`: prompts.

## Limitations

- The content is educational and may omit patient-specific considerations.
- Answers should be reviewed before public distribution.
