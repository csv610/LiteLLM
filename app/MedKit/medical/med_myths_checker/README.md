# Medical Myth Checker

This module evaluates common medical claims framed as myths and returns a structured explanation.

## Files

- `medical_myth_checker.py`: checking logic.
- `medical_myth_checker_cli.py`: CLI interface.
- `medical_myth_checker_models.py`: schemas.
- `medical_myth_checker_prompts.py`: prompts.

## Limitations

- Myth framing can oversimplify nuanced evidence.
- Statements should still be checked against current medical sources.
