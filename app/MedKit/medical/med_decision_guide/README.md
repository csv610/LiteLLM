# Medical Decision Guide

This module generates structured decision guides for symptoms or clinical questions.

## Files

- `medical_decision_guide.py`: generation logic.
- `medical_decision_guide_cli.py`: CLI interface.
- `medical_decision_guide_models.py`: schemas.
- `medical_decision_guide_prompts.py`: prompts.
- `visualize_decision_guide.py`: visualization helper.

## Limitations

- Generated decision paths are not validated clinical pathways.
- The module should be used for education or prototyping, not unsupervised triage.
