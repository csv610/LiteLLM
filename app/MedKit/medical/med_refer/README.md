# Medical Referral

This module suggests appropriate medical specialties for a presentation or question.

## Files

- `med_refer.py`: referral logic.
- `medrefer_cli.py`: CLI interface.
- `medrefer_prompts.py`: prompts.

## Limitations

- Referral recommendations are simplified and may not reflect local care pathways.
- Emergency referral decisions should not rely on generated output.
