# Anatomy Graph

This module builds structured graph-style representations for anatomical concepts and relationships.

## Files

- `anatomy_graph.py`: graph generation logic.
- `anatomy_models.py`: schemas.
- `anatomy_prompts.py`: prompts.
- `build_anatomy.py`: build helper.

## Limitations

- Graph structure depends on model interpretation.
- The output is useful for exploration, not as a validated ontology.
