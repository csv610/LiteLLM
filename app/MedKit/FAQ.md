# MedKit FAQ

## Which model should I use by default?

Use `ollama/gemma3` for local/offline-friendly workflows, or a hosted model if you need higher quality.

## Can MedKit be used for clinical diagnosis?

No. MedKit output is for reference and educational support. Do not use it as a standalone clinical decision source.

## How do I test a module quickly?

Run the corresponding CLI with `--help` first, then execute a minimal example command from that module's README.

## Where are legal and safety constraints documented?

See each module's `contract.md` file when present, and review the main MedKit README disclaimers.
