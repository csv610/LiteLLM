# Diagnostic Devices

This module generates structured information about medical diagnostic or monitoring devices.

## Files

- `medical_test_devices.py`: generation logic.
- `medical_test_devices_cli.py`: CLI interface.
- `medical_test_devices_models.py`: schemas.
- `medical_test_devices_prompts.py`: prompts.

## Limitations

- Device specifications and regulatory status can change over time.
- Output should be checked against current manufacturer or regulatory information.
