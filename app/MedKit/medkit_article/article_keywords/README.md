# Article Keywords

This module extracts keywords and terms from a medical article.

## Files

- `keyword_extraction.py`: extraction logic.
- `cli.py`: CLI interface.
- `models.py`: schemas.
- `prompts.py`: prompts.

## Why It Matters

Keyword extraction is useful for indexing, study notes, and downstream search workflows.

## Limitations

- Extracted terms may reflect prompt bias rather than controlled vocabulary standards.
- Manual review is recommended before reuse.
