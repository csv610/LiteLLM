# Mental Health

This package contains mental-health-oriented chat and assessment helpers.

## Files

- `mental_health_assessment.py`: assessment logic.
- `mental_health_chat.py`: chat workflow.
- `mental_health_chat_app.py`: CLI/app entrypoint used by `medkit-mental`.
- `mental_health_report.py`: report helper.

## Limitations

- The package is not a crisis-response service or clinical diagnostic system.
- High-risk mental-health situations require immediate human support.
